# Structs

Rust has three struct types: a classic C struct, a tuple struct, and a unit struct.

This section grows from the basics to more advanced struct patterns:

- defining regular, tuple, and unit structs,
- creating instances with field init shorthand and struct update syntax,
- adding associated functions and methods,
- choosing between owned fields, borrowed fields, `&self`, `&mut self`, and `self`,
- deriving common traits,
- destructuring nested structs,
- using consuming builder-style methods,
- writing generic structs,
- storing references in structs with lifetimes.
